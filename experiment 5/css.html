Three types of css style

CSS (Cascading Style Sheets) is used to style and layout web pages, controlling aspects like colors, fonts, and spacing.

Inline Css

  • These are defined directly in the HTML elements using the style attribute.

Internal Css

  • These are defined within a <style> tag in the <head> section of the HTML document.

External Css

  • These are defined in a separate CSS file, which is linked to the HTML document using a <link> tag in the <head>
